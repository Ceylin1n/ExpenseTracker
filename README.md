# Expense Tracker
A simple Android Expense Tracker application developed with Java in Android Studio.

## Features
- User Login & Registration
- SharedPreferences Authentication
- Income & Expense Tracking
- Balance Calculation
- RecyclerView Transaction History
- Floating Action Button
- Material Design UI

## Technologies
- Java
- Android Studio
- RecyclerView
- SharedPreferences
- Fragments
- Material Components
